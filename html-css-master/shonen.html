En Occident, le mot shōnen (少年?, qui signifie « adolescent » en japonais) est utilisé pour désigner un type de manga, le shōnen (少年漫画?, le manga pour jeune garçon, parfois écrit shounen en wāpuro rōmaji), qui est un type de manga dont la cible éditoriale est avant tout constituée de jeunes adolescents de sexe masculin. Il est à opposer au shōjo manga (少女漫画?, le manga pour jeune fille).
